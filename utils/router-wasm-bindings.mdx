# Overview
The router-wasm-bindings is a Rust crate. It contains all the custom types and messages such as OutboundBatchRequests and Sudo messages. These custom types and messages are required to support cross-chain functionality in cosmwasm contracts on the router chain. 

# Understanding the wasm bindings
Refer to [this doc](https://crates.io/crates/router-wasm-bindings) to understand more on the Router Wasm Bindings.
